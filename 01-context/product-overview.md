#Product Overview

## JPetStore

## Es una aplicacion web de tipo e-commerce

## El sistema se utiliza para la compra de productos de forma fácil y rápida, permitiendo explorar opciones sin tener que ir a la tienda físicamente y así mejorar la experiencia de compras para los clientes.

## Ayuda a la empresa a vender productos de manera digital, mejorando el alcance de clientes y generando ingresos mediante la automatización de ventas

## Usuarios principales
    Clientes
    Administradores del sistema
    Personal operativo
## Funcionalidades principales
    Ver categorías
    Listar productos
    Agregar productos al carrito
    Confirmar pedido
    Seleccionar método de pago
    Finalizar compra
